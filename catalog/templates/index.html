Mohr's Circle Calculator Home

Welcome! This is Mohr's circle calculator developed by Ms360 Group B.

Pleaso note that this program treats only in plane stress/strain condition.

Calculator Modes

Please select the mode of this program.

  • [Mode 1] Calculates the state of stress when axis is rotated.
  • [Mode 2] Calculates the required angle of rotation to get principal stress.
  • [Mode 3] Calculates the required angle of rotation to get maximum shear stress.
  • [Mode 4] Calculates the state of strain when axis is rotated.
  • [Mode 5] Calculates the required angle of rotation to get principal strain.
  • [Mode 6] Calculates the required angle of rotation to get maximum shear strain.
